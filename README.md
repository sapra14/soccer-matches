# Upcoming Soccer Matches

A simple web application that displays upcoming soccer matches fetched from a backend API.  
Users can select a date to see all scheduled matches for that day.


## Features

- Select a date to view matches on that day  
- Responsive and modern UI with CSS grid  
- Fetches live match data from a Node.js backend API


## Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Data Source: TheSportsDB API
